Colophon · Methodology

2020년 1월부터 2023년 5월까지의 거시·자산 가격 데이터를 스토리 단위로 재구성한 데이터 저널리즘 실험. 각 사건은 Prologue — Act 1–5 — Summary의 7프레임 구조를 가지며, 독립적으로 읽을 수 있도록 설계되었다.

차트는 Python matplotlib으로 프리렌더링된 PNG를 사용하며, 본문은 한국어 신문 기사 톤(평서체)을 따른다. 수치와 고유명사는 본문에 적어도 한 번 노출해 독자의 기억을 돕는다.
