# Pandemic Finance Project

This project analyzes financial data related to the pandemic.

## Structure

- `notebooks/`: Jupyter notebooks for data collection, preprocessing, and analysis.
- `data/`: Raw, processed, and web data.
- `utils/`: Utility modules for data loading, preprocessing, and visualization.
- `web/`: Web-related components (Phase 2).
- `requirements.txt`: Python dependencies.
- `.gitignore`: Git ignore file.
- `README.md`: This file.
